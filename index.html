    Pyspectrum api reference

    The pyspectrum module allows communication with usb spectrometer from python code

    Spectrometer

    High-level class to work with different spectrometers. Currently only Usb spectrometer is supported

    __init__(device, pixel_start=0, pixel_end=4096, pixel_reverse=False, dark_signal_path='dark_signal.dat')

    Parameters:

    Name Type Description Default
    device internal.RawSpectrometer

    Low-level device object

    		 required
    pixel_start int

    Index of the first usable diode

    		 0
    pixel_end int

    Index of the last usable diode

    		 4096
    pixel_reverse bool

    If True, diode order in frames will be reversed

    		 False
    dark_signal_path str

    Default path to load and save dark signal

    		 'dark_signal.dat'
